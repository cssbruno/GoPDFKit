# Static Test Assets

These files are fixtures for examples and tests.

## Fonts

The `Noto*.ttf` files were downloaded from the Noto fonts distribution
repository. The Noto project documents that fonts under its `fonts/` directory
are licensed under the SIL Open Font License.

- `NotoNaskhArabic-Regular.ttf`
- `NotoSansDevanagari-Regular.ttf`
- `NotoSansHebrew-Regular.ttf`
- `NotoSansSymbols2-Regular.ttf`

Sources:

- https://github.com/notofonts/notofonts.github.io
- https://cdn.jsdelivr.net/gh/notofonts/notofonts.github.io/fonts/NotoNaskhArabic/unhinted/ttf/NotoNaskhArabic-Regular.ttf
- https://cdn.jsdelivr.net/gh/notofonts/notofonts.github.io/fonts/NotoSansDevanagari/unhinted/ttf/NotoSansDevanagari-Regular.ttf
- https://cdn.jsdelivr.net/gh/notofonts/notofonts.github.io/fonts/NotoSansHebrew/unhinted/ttf/NotoSansHebrew-Regular.ttf

## Text

`text/unicode-edge-cases.txt` is a small handwritten fixture. It is inspired by
Unicode normalization, bidirectional text, line breaking, text segmentation, and
emoji sequence references, but it does not vendor the full Unicode test data
files.

Sources:

- https://www.unicode.org/Public/UCD/latest/ucd/NormalizationTest.txt
- https://www.unicode.org/reports/tr9/
- https://www.unicode.org/reports/tr14/
- https://www.unicode.org/reports/tr29/
- https://www.unicode.org/Public/emoji/latest/emoji-test.txt
